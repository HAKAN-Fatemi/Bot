# Bot
A Python bot that automatically fetches new ads from Meydoony.ir API and posts them to a Bale channel. Features include image support, Persian text formatting with emojis, retry mechanism, and persistent history tracking to avoid duplicate posts. and more...

ربات پایتونی که به‌طور خودکار آگهی‌های جدید را از API سایت میدونی دریافت کرده و در کانال بله منتشر می‌کند. قابلیت‌ها شامل پشتیبانی از تصاویر، فرمت‌دهی متن فارسی با ایموجی، مکانیزم تلاش مجدد، ذخیره‌سازی تاریخچه برای جلوگیری از ارسال آگهی‌های تکراری و... است.
